# StatCompare - Equipment Stat Comparison Tool
StatCompare is a tool for comparing equipment stats in games, supporting multi-language localization to help players more intuitively understand the attribute changes brought by equipment and buffs.
## Features
* Supports parsing and display of equipment attributes and buff effects
* Multi-language support (Simplified Chinese, Traditional Chinese, English)
* Customizable displayed stat types
* Minimap button for quick access
* Can show own stats when viewing other players' attributes
## Supported Stat Types
Includes parsing and comparison of various common game attributes:
* Primary stats: Strength, Agility, Stamina, Intellect, Spirit
* Defense stats: Armor, Block, Dodge, Parry, Damage Reduction
* Resistances: Arcane Resistance, Fire Resistance, Nature Resistance, Frost Resistance, Shadow Resistance
* Attack stats: Attack Power, Ranged Attack Power, Hit Chance, Critical Strike
* Spell stats: Spell Damage, Healing Effect, Spell Critical Strike, Spell Hit Chance
* Others: Haste, Armor Penetration, Spell Penetration, Health/Mana Regeneration
## Localization Support
The program automatically loads the corresponding localization file based on the game's current language setting:
* Simplified Chinese (zhCN)
* Traditional Chinese (zhTW)
* English (enUS, default)
## Usage
1. After installation, the StatCompare icon will appear on the minimap
2. Click the icon to open the configuration window
3. Select the stats you want to display in the configuration window
4. You can set options such as minimap button position, show/hide, etc.
5. The stat comparison window will automatically display when viewing other players' information
