# StatCompare - 装备属性比较工具
StatCompare 是一款用于游戏中装备属性比较的工具，支持多语言本地化，帮助玩家更直观地了解装备和 buff 带来的属性变化。
## 功能特点
* 支持装备属性与 buff 效果的解析与展示
* 多语言支持（中文简体、中文繁体、英文）
* 可自定义显示的属性类型
* 小地图按钮快捷操作
* 比较查看其他玩家属性时可同时显示自身属性
## 支持的属性类型
包含多种常见游戏属性的解析与比较：
* 主属性：力量、敏捷、耐力、智力、精神
* 防御属性：护甲、格挡、躲闪、招架、物理免伤
* 抗性：奥术抗性、火焰抗性、自然抗性、冰霜抗性、暗影抗性
* 攻击属性：攻击强度、远程攻击强度、命中率、致命一击
* 法术属性：法术伤害、治疗效果、法术暴击、法术命中
* 其他：急速、护甲穿透、法术穿透、生命 / 法力回复
## 本地化支持
程序会根据游戏当前语言设置自动加载对应的本地化文件：
* 中文简体（zhCN）
* 中文繁体（zhTW）
* 英文（enUS，默认）
## 使用方法
1. 安装后，小地图会显示 StatCompare 图标
2. 点击图标可打开配置窗口
3. 在配置窗口中可选择需要显示的属性
4. 可设置小地图按钮位置、显示 / 隐藏等选项
5. 查看其他玩家信息时可自动显示属性比较窗口
